**Compilando CSS y Javascript**

**Bootstrap se compila con Sass (no con CSS plano). Debes tener instalado:**
* **Node.js**
* **npm**
* **sass o herramientas como webpack**

**Comando para compilar:**

```
sass scss/bootstrap.scss css/bootstrap.css
```
